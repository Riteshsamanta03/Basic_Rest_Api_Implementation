# Knowledge Hub

A basic project demonstrating the implementation of **RESTful APIs with CRUD operations** using **Node.js, Express, and EJS**.

---

##  Project Overview

Knowledge Hub is a simple post-based application where users can:

* Create new posts 
* View all posts 
* Edit existing posts 
* Delete posts 

This project focuses on understanding backend fundamentals like routing, HTTP methods, and REST architecture.

## Features

* RESTful API structure
* CRUD operations (Create, Read, Update, Delete)
* Dynamic rendering using EJS
* Clean UI with basic CSS styling
* Unique post identification using UUID
* Method override for supporting PATCH & DELETE

---

## Concepts Implemented

* RESTful Routing
* HTTP Methods (GET, POST, PATCH, DELETE)
* Express.js Routing & Middleware
* EJS Templating Engine
* UUID for unique IDs
* Method-Override for advanced HTTP requests

## How to Run

1. Clone the repository

```
git clone https://github.com/your-username/your-repo-name.git
```

2. Install dependencies

```
npm install
```

3. Run the server

```
node index.js
```

4. Open in browser

```
http://localhost:3000/posts
```
